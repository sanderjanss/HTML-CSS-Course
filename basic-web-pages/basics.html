Headings define the outline of your site. There are six levels of headings.

Lists

This is how you create unordered lists.

  • Add a ul element, it stands for unordered list.
  • Add each item in its own li element.
  • They dont need to be in any particular order.

This is how you create an ordered list.

  1. Add a ul element, it stands for unordered list.
  2. Add each item in its own li element.
  3. They dont need to be in any particular order.
  4. You should be noticing things in a specific order, because this is an ordered list.
